# paperclip-mcp

MCP (Model Context Protocol) server for the [Paperclip](https://github.com/paperclipai/paperclip) control-plane API.

Exposes the Paperclip REST API (`/api`) as MCP tools so agents can manage companies, agents, issues, budgets, and approvals over MCP.

> Status: scaffolding. Server implementation TBD — see API reference at https://docs.paperclip.ing/reference/api/overview.

## Development

This repo is a submodule of [`paperclip-development-stack`](https://github.com/git-clone-abhinav/paperclip-development-stack), alongside the `paperclip` app and `companies` catalog.
